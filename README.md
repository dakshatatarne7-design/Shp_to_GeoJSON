# Shp_to_GeoJSON
Conversion of shapefile into GeoJSON by Python Automation
In Google Colab ESRI Shapefile is converted into GeoJSON.
ESRI shapefile is the multi-file weighted file used in offilne GIS workflow for spatial analysis, while GeoJSON lightweighted file format used in online GIS environment to perform spatial analysis.
Reference: https://fme.safe.com/guides/spatial-computing/shapefile-to-geojson/
